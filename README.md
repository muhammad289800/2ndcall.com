# Non-VoIP Number Manager - Comprehensive number search, purchase, and management system with SMS/call capabilities and multi-provider support.

Features:
- Search Numbers
- Buy Numbers
- Manage Numbers
- SMS & Call
- Provider Setup

Environment Variables:
- TELNYX_API_KEY
- TELNYX_CONNECTION_ID
- TWILIO_ACCOUNT_SID
- TWILIO_AUTH_TOKEN
